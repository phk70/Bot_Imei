Добрый день.
После клонирования https://github.com/phk70/Bot_Imei.git
создаем .env и устанавливаем зависимости.

Запуск с двух файлов:
app.py - Flask.
bot.py - запуск Бота.


По функционалу:
При запуске бота создается простая БД Users.db
Запись пользователя в БД происходит при нажатии на кнопку Старт в боте. 
При записи так же формируется token библиотекой secret.

У всех пользователей нет никаких прав.
Для получения прав админа вводим в поле ввода пароль указанный в .env (ADMIN_PASSWORD).
После этого можно будет давать права пользователям на пользование ботом через функцию open_permission_user либо удалять пользователей.
Ввод IMEI и Telegram_id для удаления и добавления реализован через Состояния.
Таким образом реализовал белый список.

У простого пользователя из клавиатуры только проверка IMEI и затем выбор сервисов.

API прописаны через Flask
При отправке тестовых запросов через Postmamn возвращают JSON.
